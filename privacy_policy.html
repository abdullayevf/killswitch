KillSwitch Privacy Policy

Effective date: March 2, 2026 · Last updated: March 2, 2026

KillSwitch is a Chrome Extension that generates tailored freelance proposals for job posts on Upwork, Fiverr, and Freelancer. This policy explains what data the extension handles and how it is stored and used.

Data we do NOT collect

  • We do not operate servers or backends of any kind.
  • We do not collect, transmit, or store any data on our own infrastructure.
  • We do not use analytics, crash reporting, or any third-party tracking SDKs.
  • We do not track usage, clicks, or behavior of any kind.

API keys

Your API keys for Groq, Google Gemini, or Hugging Face are stored exclusively in chrome.storage.sync, which is managed by Chrome and optionally synced across your own signed-in Chrome profile. Keys are never transmitted to KillSwitch servers (which do not exist) and are only sent directly to the AI provider you have configured when you generate a proposal.

Job descriptions and proposals

When you click "Generate Proposal," the job description you entered is sent directly from your browser to the AI provider you selected (Groq, Google Gemini, or Hugging Face). This request is made using your own API key and is governed by that provider's privacy policy. KillSwitch does not see, log, or retain the contents of these requests.

Feedback and learning history

When you rate a proposal with 👍 or 👎, the proposal text (truncated to 600 characters) and your rating are stored locally in chrome.storage.local on your device only. This data is used solely to personalize future proposals via few-shot prompt injection — it is never transmitted anywhere. You can delete all feedback history at any time from the Settings page.

Permissions

KillSwitch requests the following Chrome permissions and uses them as described:

  • contextMenus — Adds a right-click context menu item on Upwork, Fiverr, and Freelancer pages. When you select text from a job description and right-click, the menu sends that text to the extension popup to pre-fill the job description field.
  • tabs — Used solely to open the Settings page in a new tab when you click the settings icon in the popup. No tab content is read or modified.
  • storage — Stores your API keys and preferences (tone, goal, language defaults, custom prompt) in chrome.storage.sync, and your local proposal feedback history in chrome.storage.local. Nothing is sent to any server operated by this extension.
  • Host permissions (Upwork, Fiverr, Freelancer, AI provider domains) — Required for two purposes: (1) to inject the content script into freelance platform pages for textarea auto-fill; (2) to make direct API calls from the extension to your configured AI provider (Groq, Google Gemini, or Hugging Face) using your own API key.

No remote code execution

This extension does not execute remote code. All JavaScript is bundled at build time and included in the extension package. The extension makes standard fetch() calls to AI provider REST APIs and receives text responses — no code is fetched from or executed from any remote source.

Third-party AI providers

Requests to AI providers are subject to their own privacy policies:

  • Groq Privacy Policy
  • Google Privacy Policy
  • Hugging Face Privacy Policy

Clearing your data

You can remove all data stored by KillSwitch at any time:

  • Open the KillSwitch Settings page and click "Clear Learning History" to delete all feedback.
  • Remove your API key from the Settings page to delete stored credentials.
  • Uninstalling the extension removes all locally stored data from your device.

If you have questions about this policy, please open an issue on the KillSwitch GitHub repository.
